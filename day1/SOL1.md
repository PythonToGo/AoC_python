# Day 1

## 문제 요약
- 다이얼: 0~99 원형
- 시작 위치: 50
- 회전 명령: L/R + 거리
- L: 왼쪽(낮은 숫자), R: 오른쪽(높은 숫자)

## Part 1
- 각 회전 **후** 다이얼이 0을 가리키는 횟수 세기
- 회전 후 최종 위치만 확인

### 구현
- `pos = 50` 초기화
- 각 라인 파싱: `direction` (L/R) + `distance` (숫자)
- L이면 `(pos - distance) % 100`
- R이면 `(pos + distance) % 100`
- 회전 후 `pos == 0`이면 카운트 증가

## Part 2
- 회전 **중** 모든 클릭에서 0을 가리키는 횟수 세기
- 회전 과정의 모든 단계 확인 필요

### 구현
- `pos = 50` 초기화
- 각 회전을 `distance`만큼 한 클릭씩 진행
- `step = -1` (L) 또는 `1` (R)
- 각 클릭마다 `pos = (pos + step) % 100`
- 매 클릭마다 `pos == 0`이면 카운트 증가

## 키포인트
- 모듈로 연산 `% 100`으로 원형 처리
- Part 1: 회전 후만 확인
- Part 2: 회전 중 모든 단계 확인
